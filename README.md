## EGO Project - Frontend

#### 1. Clone project

```bash
$ git clone https://github.com/Enclave-EGO/frontend.git
```

#### 2. Change directory to folder frontend

```bash
$ cd frontend
```

#### 3. Install dependencies

```bash
$ npm install
```

#### 4. Run in developement mode

```bash
$ npm start
```

### Started website: http://localhost:3000/
